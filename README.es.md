# Remote Connector — Sistema de Administración Remota Distribuida

Sistema listo para producción para administración remota de nodos distribuidos. El servidor expone un dashboard web y recibe conexiones WebSocket seguras (WSS) de los nodos cliente. Desde el dashboard puedes monitorear métricas en tiempo real, ejecutar comandos remotos y gestionar las conexiones de clientes.

[🇬🇧 Read in English](README.md)

---

## Características

- **Dashboard Web**: SPA React + Vite con métricas en tiempo real, ejecución de comandos y gestión de clientes
- **WebSocket Seguro**: Conexiones WSS con autenticación JWT entre servidor y clientes
- **Métricas en Tiempo Real**: Monitoreo de CPU, RAM y disco con gráficos históricos
- **Ejecución de Comandos Remota**: Envía comandos de shell a cualquier cliente conectado y recibe la salida
- **JWT Dual**: Tipos de token separados para usuarios del dashboard (24h) y nodos cliente (30d)
